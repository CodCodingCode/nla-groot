# Audit report outputs

Markdown and JSON written by `scripts/eval/audit_*.py` (diversity, bullet informativeness, prompt hardening, etc.).

These files are **regenerated** by running the scripts from the repo root; they are not hand-maintained sources of truth.

Historical filenames (`agent2_*`, `agent3_*`, …) are kept for backward compatibility with `docs/sft_plan/v4_training_recon_audit.md` and repair notes under `v4_repair/`.
